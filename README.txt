Island of Life — Biology Simulation Game
==========================================

HOW TO RUN
----------
1. Make sure Node.js is installed on your computer.
   To check, open a terminal and run:  node --version
   If you see a version number (v14 or higher), you're good.
   If not, download Node.js from https://nodejs.org (use the LTS version).

2. Unzip this file if you haven't already.

3. Open a terminal in the unzipped folder.
   (On Windows: right-click the folder → "Open in Terminal"
    On Mac: right-click the folder → "New Terminal at Folder")

4. Run:  node run.js

5. Your browser will open automatically at http://localhost:3000

6. Press Ctrl+C in the terminal when you are done.

No login required. No internet connection needed after first load.
Your islands are saved automatically in the browser.


WHAT'S INSIDE
-------------
This is a biology simulation game built as a high school final project.
It demonstrates concepts in Evolution, Ecology, and DNA through a real-time
god-simulation where species arrive, compete, adapt, and evolve on a living island.

Click "CLICK THIS IF YOU ARE MR. FOSTER" on the islands page for the
showcase view with biology concept explanations.


TROUBLESHOOTING
---------------
"Port 3000 is already in use" — close other programs and try again,
or visit http://localhost:3000 directly in your browser after the error.

Browser didn't open automatically — manually go to http://localhost:3000.
